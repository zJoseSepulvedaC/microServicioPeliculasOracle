ojdbc8-full.tar.gz - JDBC Thin Driver and Companion JARS
========================================================
This TAR archive (ojdbc8-full.tar.gz) contains the 23.7.0.25.01 release of the Oracle JDBC Thin driver(ojdbc8.jar), the Universal Connection Pool (ucp.jar) and other companion JARs grouped by category. 

(1) ojdbc8.jar (7196115 bytes) - 
(SHA1 Checksum: 65cc9902e2d3f270908f7e45b9570079988d1b5f)
Oracle JDBC Driver compatible with JDK8, JDK11, JDK12, JDK13, JDK14, and JDK15.

(2) ucp.jar (1503025 bytes) - (SHA1 Checksum: 0ae7c39ebf256b8fa360788229488949bd4f4ac9)
Universal Connection Pool classes to be used with ojdbc8.jar -- for performance, scalability, high availability, sharded and multitenant databases.
 
(3) rsi.jar (260433 bytes) - (SHA1 Checksum: f9a781869b478040020ab26ab3f4d2312f49ab5a)
Reactive Streams Ingestion (RSI) 

======================
Security Related JARs
======================
Java applications require some additional jars to use Oracle Wallets. 
You need to use all the three jars while using Oracle Wallets. 

(4) oraclepki.jar (502640 bytes) - (SHA1 Checksum: 57a2d44f2a2a69cac5733c2963f33e207054cd32)
Additional jar required to access Oracle Wallets from Java

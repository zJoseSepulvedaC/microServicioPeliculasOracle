Microservicio Películas

Este microservicio REST desarrollado en Spring Boot permite gestionar información de películas almacenadas
en una base de datos Oracle Autonomous en la nube. Actualmente, admite operaciones de lectura (GET) y creación (POST) de
 registros de películas.

 Tecnologías Utilizadas

    Java 21

    Spring Boot

    Spring Web

    Oracle JDBC (ojdbc11)

    Base de Datos Oracle Autonomous

    Maven

 Endpoints Disponibles

GET /peliculas

Obtiene todas las películas registradas.
GET /peliculas/{id}

Busca una película por su ID.
POST /peliculas

Agrega una nueva película.

 Estructura de la tabla peliculas

CREATE TABLE PELICULAS (
  ID        NUMBER GENERATED BY DEFAULT AS IDENTITY,
  TITULO    VARCHAR2(200),
  ANIO      NUMBER(4),
  DIRECTOR  VARCHAR2(200),
  GENERO    VARCHAR2(100),
  SINOPSIS  CLOB,
  PRIMARY KEY (ID)
);

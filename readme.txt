# Microservicio Películas

Este microservicio REST desarrollado en **Spring Boot** permite gestionar información de películas almacenadas en una **base de datos Oracle Autonomous** en la nube. Actualmente, admite operaciones de lectura (**GET**) y creación (**POST**) de registros de películas.

## Tecnologías Utilizadas

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Oracle JDBC (ojdbc8)
- Base de Datos Oracle Autonomous
- Maven

## Endpoints Disponibles

### GET `/peliculas`

Obtiene todas las películas registradas en la base de datos.

### GET `/peliculas/{id}`

Busca una película por su identificador (`id`).

### POST `/peliculas`

Agrega una nueva película enviando los datos en formato **JSON**.

**Ejemplo de body:**

```json
{
  "titulo": "Inception",
  "anio": 2010,
  "director": "Christopher Nolan",
  "genero": "Ciencia Ficción",
  "sinopsis": "Un ladrón experto en robar secretos del subconsciente es reclutado para una tarea especial."
}
```

## Estructura de la tabla `PELICULAS`

```sql
CREATE TABLE PELICULAS (
  ID        NUMBER GENERATED BY DEFAULT AS IDENTITY,
  TITULO    VARCHAR2(200),
  ANIO      NUMBER(4),
  DIRECTOR  VARCHAR2(200),
  GENERO    VARCHAR2(100),
  SINOPSIS  CLOB,
  PRIMARY KEY (ID)
);
```
